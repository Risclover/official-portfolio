Initial Concept

As I mentioned earlier, I want a blog on my official Portfolio site. In a nutshell, I'm planning on using this blog to create "Project Breakdowns" (in-depth looks at how I create my various projects) and posts about me learning how to program. Maybe someday, when I feel like I'm more of a professional, I'll start posting tutorials, guides, opinion pieces, etc. For now, my blog is all about me learning.

I almost created a blog via Wix. I've used their site builder in the past (back before I started coding). However, while I was working on creating that blog, I thought to myself, "What am I doing? I know how to use HTML and CSS extremely well, and I'm getting the web part of JavaScript down, too." And thus, I decided to code my own blog.

Update - Golden Ratio

So I've done a bit of editing to my blog mockup. I fell into a bit of a Google search rabbit hole a couple of hours ago - beginning with researching why the majority of websites use some variation of gray as their font color instead of black. The search for "gray text not black" generated this as a result: 7 Deadly Web Design Sins You Might Be Making. Color me intrigued. I clicked on it and read it (decent article!).

A little over halfway into the article, the author mentions the Golden Ratio, which is something I'd never actually heard of, as far as my present memory knows. Specifically, he suggests using the Golden Ratio to design certain aspects of your website, such as your font's line height. Some side googling led me to multiple sources reporting that setting your line height at 168% not only sets your current font size to the "golden ratio line height", but any size, a la responsive web design. (Yay percentages!)

I continued to read up on the Golden Ratio, and researched using it in web design. I don't have a "natural knack" for web design (or visual art, for that matter), so any additional help I can get is always appreciated. I am a sponge waiting to absorb all of your tips, tricks, and knowledge. Hydrate me.

Comparing before Golden Ratio and after Golden Ratio, the version with the Golden Ratio applied does look even better! I'm very impressed!

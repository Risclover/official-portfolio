Introduction (First Blog Post!)

09 February, 2022

Personal

Hello! Thanks for visiting my personal website. My name is Sara Dunlop, and I am an aspring computer programmer.

My late father was a computer programmer, as is my uncle (his brother). I grew up on a computer and, when I was a teenager, I dabbled briefly in web development.

For the record, I'm 29 now.

Much more recently, although I wasn't necessarily trying to learn any coding languages, I was very involved with trying to create tools for some of the games I liked to play. I used Google Sheets (Google's spreadsheets program, very similar to Excel) to create these tools. Although I'm sure you wouldn't necessarily call it a "coding language", and although I am very aware that it isn't the same thing necessarily, I do think that what I did on Google Sheets is similar to coding with a real language.

Over the last year, I gained a strong interest in learning real programming languages. I know HTML and CSS very well, and JavaScript was my first real language. Although it wasn't exactly easy at first, I do think that I know enough JavaScript to be pretty confident in using it.

Next up, I'd like to learn PHP and SQL so that I can use them to reference databases on websites, build blogs, create tools, and more. I would also like to focus on learning Python.

I plan to use this blog for talking about my journey in learning how to code and becoming a computer programmer. I also want to use it for discussing my projects at length. Beyond that, you may find opinion pieces on here related to coding (or my life). I hope you enjoy reading pieces straight from my brain!

This is my portfolio, where I showcase all of the projects I'm proud of. If you hover over an image and click on the "View" icon (with an eye icon), you'll find additional information including the languages, frameworks, and tools used to create the project, and links to the project's blog post (where I discuss the code), the Github repo containing the project's source code, and a live demo of the project itself. If you have any questions or comments about a project, feel free to message me via the Contact page!

Calculator

A browser-based calculator app, including a decimal key, clear, delete, an exponent key, and the capability to change between positive and negative.

Etch-a-Sketch

A drawing app in a similar style as an etch-a-sketch. Allows you to select your color(s) and change the grid size, and includes various modes including Party Mode, Eraser Mode, and Shading.

Product Landing Page

A fully-responsive product landing page. Includes several sections, each with beautifully-styled elements that fit the theme.

Number-Guessing Game

A number guessing game in which the player has to guess the number. After a guess, the computer lets you know whether you were too high or too low. Try to guess the number with as few guesses as you can.

A Tribute to Muse

A tribute page to the band Muse. Completed as part of the freeCodeCamp course, this is the first webpage I created where I ended up very proud of my result.
